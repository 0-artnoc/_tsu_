```shell,tsu
    tsu A su interface wrapper for Termux

    Usage:
        tsu
        tsu [ -s SHELL ]  [-pe] [USER]
        tsu --debug [ -s SHELL ]  [-pel] [USER]
        tsu -h | --help | --version


    Options:
    -s <shell>   Use an alternate specified shell.
    -l           Start a login shell.
    -p           Prepend system binaries to PATH
    -h --help    Show this screen.

```
